  ✓ Secure login (Name + Roll + Password — no supervisor access)
  ✓ Find & filter supervisors by DIU department (CSE, SWE, ICT, ICE…)
  ✓ Send supervision request with personal message
  ✓ Browse 50 project ideas across 12 CS/Engineering categories
  ✓ Save/bookmark projects (tracked per session)
  ✓ Watch 15 real YouTube videos (embedded player)
  ✓ Track request status (Pending / Accepted / Rejected)

  ✓ Available Slots card → manage slot capacity
  ✓ View detailed student profiles
  ✓ Real-time request accept/reject with notifications

ADMIN MODULE:
  ✓ System dashboard with analytics
  ✓ User management (add/remove students & supervisors)
  ✓ Project topic management (add/edit/delete)
  ✓ Category management
  ✓ View all requests across the system
  ✓ Feedback management
